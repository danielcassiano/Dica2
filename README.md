## 🔁 Como contribuir (PASSO A PASSO)

### 1. Fazer Fork
No GitHub, clique em **Fork** no repositório original.

---

### 2. Clonar o Fork

git clone https://github.com/SEU-USUARIO/node.js-e-express.git cd node.js-e-express

---

### 3. Criar uma branch

git checkout -b "Seu Nome Completo"

---

### 4. Criar pasta da atividade
- Vespertino:

mkdir -p "Atividades-A1/Vespertino/Seu Nome Completo"

- Noturno:

mkdir -p "Atividades-A1/Noturno/Seu Nome Completo"

---

### 5. Adicionar arquivos

git add .

---

### 6. Commit

git commit -m "A1: Atividade de [Seu Nome Completo] - [Turno]"

---

### 7. Push

git push origin "Seu Nome Completo"

---

### 8. Pull Request
No GitHub, clique em:
**Compare & pull request**

---

## ⚠️ Importante

- Não é possível dar push direto no repositório original (erro 403)
- Use sempre **Fork + Pull Request**

---

## 🔥 Fluxo

Fork → Clone → Branch → Alterar → Add → Commit → Push → Pull Request
